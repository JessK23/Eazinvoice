# EazInvoice WordPress Pro Product

## Monetization Model

Use a freemium model:

- Submit the basic plugin to the official WordPress plugin directory to drive awareness and organic traffic.
- Sell EazInvoice WordPress Pro independently from `eazinvoice.com`.
- Position Pro as a paid tier benefit for Standard, Pro, and Business EazInvoice subscribers.

## Recommended Distribution

Sell from the EazInvoice website as a digital product using:

- EazInvoice's own subscription checkout once billing is live, or
- WooCommerce + a software license/download extension, or
- Easy Digital Downloads, or
- Lemon Squeezy/Gumroad-style checkout if a lighter setup is preferred.

The recommended first implementation is EazInvoice's own pricing/subscription page once available. WooCommerce can still be used as a temporary sales backend if it is faster to launch.

## Product Name

EazInvoice WordPress Pro

## Product Positioning

Turn WordPress service requests, B2B billing actions, customer inquiries, and optional store orders into GST-ready EazInvoice workflows.

## Paid Features

- Lead form to invoice draft
- Service inquiry to quotation or invoice
- B2B customer profile mapping
- Payment status sync when gateway payment is captured
- Invoice PDF/download link display
- Purchase Order / Work Order support
- GST-ready revenue and payment reporting
- Optional WooCommerce order-to-invoice sync
- Priority support
- License-key based plugin updates

## Suggested Pricing

- Standard tier: INR 199/month, billed yearly as INR 2,388. WordPress paid access included for one website.
- Pro tier: INR 499/month, billed yearly as INR 5,988. AI invoice/PO/report features and WordPress paid access for up to three websites.
- Business tier: INR 999/month, billed yearly as INR 11,988. Multi-site access, team workflow, API access, SMTP/gateway controls, and priority support.

How The WordPress Plugin Works

The free plugin automatically adds an EazInvoice action button to the WordPress site. Paid tiers can later add lead-to-invoice, B2B, PO/WO, reports, accounting snapshots, and optional WooCommerce workflows.

1Save Settings

Add account email, workspace URL, and button label.

2Auto Site Button

Show EazInvoice automatically on the public website.

3Open Workspace

Click the button and open the connected workspace.

4Upgrade

Paid tiers can unlock service, B2B, PO/WO, and optional WooCommerce automation.

# EazInvoice Web Application User Manual

## Purpose

This manual explains how a user should operate the EazInvoice web application after signup and login.

## Login and Account Access

1. Open `https://www.eazinvoice.com/apps/web/index.html`.
2. Select **Signup / Register** for a new account, or **Login** for an existing account.
3. Use email OTP verification and password login.
4. After login, the home page should show the logged-in profile and the active plan access button.

## Profile and Business Setup

1. Open **User Access**.
2. Use **Update Profile** for individual account details.
3. Use **Business Profiles** or **Add Company** when the account needs company billing details.
4. Complete PAN, GSTIN, address, phone, email, UPI, and bank details where applicable.

## Invoice Workflow

1. Open **Invoices**.
2. Choose **Generate New Invoice**.
3. Select existing customer or add a new customer.
4. Select currency.
5. Add goods, services, HSN/SAC or tax details, quantity, rate, GST/tax, and discount amount.
6. Save as draft when incomplete.
7. Create invoice when final.
8. Generate PDF or print after creation.
9. Record payments from the invoice record until the invoice is fully paid.

## PO / WO Workflow

1. Open **Purchase / Work Order**.
2. Choose PO or WO.
3. Select or add vendor details.
4. Add goods/services, quantity, rate, tax, and delivery/payment terms.
5. Save draft when incomplete.
6. Create PO/WO when final.
7. Created PO/WO records feed expense and profit/loss reports.

## Reports

1. Open **Reports**.
2. Use month, year, custom period, or financial year filters.
3. Review revenue, invoices, expenses, PO/WO, profit and loss, accounting summary, and compliance snapshots.
4. Use export or print options where available.

## Subscription

Paid plans are shown as monthly value but collected yearly:

- Standard: INR 199/month, billed yearly INR 2,388.
- Pro: INR 499/month, billed yearly INR 5,988.
- Business: INR 999/month, billed yearly INR 11,988.

Free users see indicators for restricted paid features. Paid users inherit the previous tier features.

## Business Tier

Business users can manage:

- Sub-users and role-specific access.
- Approval requests.
- Business SMTP settings.
- Razorpay gateway settings for business invoices.
- Compliance profile and reminders.
- API keys for integrations.
- Audit trail and notification center.

## Security Notes

- Do not share OTPs, passwords, API keys, Razorpay secrets, or SMTP passwords.
- Use role-based sub-user access instead of sharing the owner login.
- Created/deleted records are preserved historically for audit integrity.

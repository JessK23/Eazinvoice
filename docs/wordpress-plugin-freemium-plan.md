# EazInvoice WordPress Plugin Freemium Plan

## Model Chosen

Freemium.

The free WordPress plugin should be submitted to the official WordPress plugin directory as a traffic and trust channel. The paid WordPress Pro plugin should be sold independently from the EazInvoice website.

## Free Plugin

Purpose: drive adoption and send WordPress users into the EazInvoice SaaS product.

Free features:

- EazInvoice admin settings page
- Account email and workspace URL fields
- Basic API key placeholder for future connection
- Invoice CTA shortcode
- EazInvoice branding and upgrade CTA

Suggested directory name:

`EazInvoice - Invoicing for MSMEs`

## Paid Plugin Product

Product name:

`EazInvoice WordPress Pro`

Paid features:

- WooCommerce order-to-invoice sync
- Customer mapping
- Payment status automation
- Invoice PDF/download links
- Purchase Order / Work Order support
- GST-ready reports
- Priority support
- License-key updates

## Distribution

Recommended first channel:

Sell from `eazinvoice.com` as a digital product using WooCommerce or Easy Digital Downloads.

This keeps sales under the EazInvoice brand and allows the website to manage:

- plugin purchases
- subscriptions
- license keys
- renewal reminders
- invoices
- upgrade offers

## Tier Mapping

- Free plan: free WordPress CTA plugin
- Standard plan: WordPress Pro for one website
- Pro plan: WordPress Pro for up to three websites
- Business plan: multi-site WordPress Pro, API access, team workflow, priority support

## Marketplace Path

1. Submit the free plugin to the WordPress.org plugin directory.
2. Add upgrade links in the free plugin admin page.
3. Sell the Pro plugin from EazInvoice website.
4. Later, list Pro on specialized marketplaces only if support and licensing operations are mature.
